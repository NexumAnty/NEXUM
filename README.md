## Why NEXUM?

Most anti-detection browsers today are black boxes.  
They promise anonymity, but never explain *how* or *why* it works.

NEXUM is different.

This project started from a simple idea:

**What if an anti-detection browser was built like real infrastructure ‚Äî not a growth hack?**

No hidden telemetry.

## Core Principles

- **Privacy-first** ‚Äî no background tracking, no hidden analytics  
- **Local-first** ‚Äî profiles are stored locally by default  
- **Isolation by design** ‚Äî each profile is a separate runtime  
- **Transparency** ‚Äî clear limits, no false guarantees  
- **Built slow, built right**
